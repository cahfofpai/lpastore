# Frontend

Install dependencies

```bash
npm install
```

Run

```bash
npm run dev
```

Build

```bash
npm run build
```

Add package as dev dependency and install it

```bash
npm install --save-dev <package>
npm i -D <package>
```

Remove package (also from package.json)

```bash
npm r <package>
```

Update style

```bash
npm run generate-css
```

Check for errors

```bash
npm run check
```
